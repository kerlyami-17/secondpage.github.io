Lenguaje JavaScript (js).

JavaScript es un lenguaje de programación interpretado que te permite implementar funciones complejas en páginas web. Cuando una página web va más allá de mostrar información estática y ofrece actualizaciones de contenido, mapas interactivos, animaciones, reproducción de videos y más, es probable que JavaScript esté involucrado. En resumen, es la tercera capa del pastel de las tecnologías web estándar, complementando HTML (para estructurar el contenido) y CSS (para aplicar estilo). JavaScript permite crear contenido dinámico y controlar multimedia, lo que lo convierte en una herramienta poderosa para desarrolladores web.
